# May 2026 (version 1.122)

## Remote - SSH

### EOL for 32-bit ARM Linux hosts

VS Code release 1.122 is the last release that supports running the prebuilt servers on 32-bit ARM Linux hosts. Starting with VS Code release 1.123, you need to use a supported x86_64 or ARM64 Linux host for remote development. For more information, see the [Remote Development FAQ](https://code.visualstudio.com/docs/remote/faq#_can-i-run-vs-code-server-on-32bit-arm-linux-hosts).
