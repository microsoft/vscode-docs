## March 18, 2026

* Add support for zooming in and out in the image carousel. [#302003](https://github.com/microsoft/vscode/issues/302003)
* The integrated browser now allows bypassing certificate errors for localhost HTTPS connections with self-signed certificates. [#298982](https://github.com/microsoft/vscode/issues/298982)
* Reveal in File Explorer now works correctly when connected to WSL. [#299491](https://github.com/microsoft/vscode/issues/299491)
* Improved screen reader labels for implicit context. [#301568](https://github.com/microsoft/vscode/issues/301568)

## March 16, 2026

* Debug launch configurations can now be hidden per-platform using the `"windows"`, `"linux"`, or `"osx"` properties in `launch.json`. [#293394](https://github.com/microsoft/vscode/issues/293394)
