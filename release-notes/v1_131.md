## July 22, 2026

* Add support for quick actions on voice buttons, so you can select a specific microphone or disable the microphone directly from the Voice Mode and Dictation controls. _[#327013: Add context menu actions (Select Microphone / Disable) to Voice Mode and Dictation mic buttons](https://github.com/microsoft/vscode/issues/327013)_

* Add support for automatically disabling extension-provided voice mode when you use the built-in voice mode, which avoids conflicting voice mode experiences. _[#327000: Built in voice mode should disable extension voice mode](https://github.com/microsoft/vscode/issues/327000)_

* Add support for selecting a folder from a quick pick in the new session view, which speeds up creating agent sessions in the correct workspace. _[#326987: Agents Window: Quick Pick for folder selection in new session view](https://github.com/microsoft/vscode/issues/326987)_

* Add support for built-in on-device dictation in the editor and terminal, and add support for read-aloud text-to-speech, so you can complete more voice workflows without external extensions. _[#326969: Extend built-in on-device dictation to editor dictation, terminal voice, and add read-aloud (TTS)](https://github.com/microsoft/vscode/issues/326969)_

* Add support for migrating prompts with triage and cleanup actions, so you can review and remove outdated prompts while moving to the new prompt experience. _[#325660: Migrate Prompts should help me triage and delete old prompts](https://github.com/microsoft/vscode/issues/325660)_

## July 21, 2026

* Add support for timestamps on older chat conversations, so you can better understand conversation history at a glance. _[#324482: Show timestamp for older conversations](https://github.com/microsoft/vscode/issues/324482)_

* Add support for disabling the terminal dimensions overlay during resize, which reduces visual noise while you adjust the terminal panel size. _[#295790: Make it possible to disable dimensions visual overlay on terminal resize](https://github.com/microsoft/vscode/issues/295790)_
