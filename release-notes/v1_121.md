## May 13, 2026

* Add support for pinning favorite models in the language model picker. [#299776](https://github.com/microsoft/vscode/issues/299776)

* Set a `VSCODE_AGENT` environment variable when Copilot Chat runs commands in the terminal. [#311734](https://github.com/microsoft/vscode/issues/311734)

* Add an "Add to Chat" option to the right-click context menu in integrated browser. [#305718](https://github.com/microsoft/vscode/issues/305718)

* Automatically dispose background terminals created by the chat agent once their command finishes. [#287177](https://github.com/microsoft/vscode/issues/287177)

* Allow extensions contributing `markdown.previewScripts` to declare their scripts as ES modules by using the `{ "path": "...", "type": "module" }` object syntax. [#316328](https://github.com/microsoft/vscode/issues/316328)

* Expand terminal tool output compression to cover more commands, including test runners (`pytest`, `jest`, `cargo test`), build tools (`tsc`, `cargo build`, `make`), linters, Docker, and package managers. [#315881](https://github.com/microsoft/vscode/issues/315881)

## May 12, 2026

* Add an idle-silence timer to the `run_in_terminal` tool that automatically promotes a sync command to background execution when it produces no output for a configurable period. [#315884](https://github.com/microsoft/vscode/issues/315884)

* Fix multi-line shell commands in the Agent Host terminal tool. [#312922](https://github.com/microsoft/vscode/issues/312922)

* Bundle a newer version of ConPTY (`conpty.dll`) directly with VS Code on Windows. [#224488](https://github.com/microsoft/vscode/issues/224488)

## May 11, 2026

* Add keyboard-interactive authentication support for Agent Host SSH connections. [#315588](https://github.com/microsoft/vscode/issues/315588)
