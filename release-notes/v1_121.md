## May 18, 2026

* Add support for sharing comment feedback in Agent Host sessions, enabling users to provide inline feedback on agent responses. [#315061](https://github.com/microsoft/vscode/issues/315061)

## May 17, 2026

* Improve inline suggestion quality when the suggest control is visible by preventing the control from overriding higher-quality inline completions. [#316899](https://github.com/microsoft/vscode/issues/316899)

## May 16, 2026

* Display a token usage widget in Agent Host sessions to show resource consumption. [#316514](https://github.com/microsoft/vscode/issues/316514)

## May 15, 2026

* Improve tool display in Agent Host sessions to show friendly tool names and enhanced input/output UI for tool calls. [#315182](https://github.com/microsoft/vscode/issues/315182)

* Add auto-approve picker for Agent Host connections. [#316179](https://github.com/microsoft/vscode/issues/316179)

* Add auto permission mode for Claude Agent sessions, using a model-based classifier to automatically approve or deny tool calls. [#316647](https://github.com/microsoft/vscode/issues/316647)

* Add an "Open in Integrated Browser" context menu option for files in the Explorer and editor tabs. [#309057](https://github.com/microsoft/vscode/issues/309057) [#315969](https://github.com/microsoft/vscode/issues/315969)

* Render Mermaid diagrams in Copilot CLI chat responses instead of displaying them as raw code blocks. [#315185](https://github.com/microsoft/vscode/issues/315185)

* Add built-in Mermaid.js diagram rendering in Markdown previews, removing the need for a third-party extension. [#293028](https://github.com/microsoft/vscode/issues/293028)

## May 14, 2026

* Automatically pre-select the current workspace folder when opening the Agents window from a VS Code instance. [#304200](https://github.com/microsoft/vscode/issues/304200)

## May 13, 2026

* Add support for pinning favorite models in the language model picker. [#299776](https://github.com/microsoft/vscode/issues/299776)

* Set a `VSCODE_AGENT` environment variable when Copilot Chat runs commands in the terminal. [#311734](https://github.com/microsoft/vscode/issues/311734)

* Add an "Add to Chat" option to the right-click context menu in integrated browser. [#305718](https://github.com/microsoft/vscode/issues/305718)

* Automatically dispose background terminals created by the chat agent once their command finishes. [#287177](https://github.com/microsoft/vscode/issues/287177)

* Allow extensions contributing `markdown.previewScripts` to declare their scripts as ES modules by using the `{ "path": "...", "type": "module" }` object syntax. [#316328](https://github.com/microsoft/vscode/issues/316328)

* Expand terminal tool output compression to cover more commands, including test runners (`pytest`, `jest`, `cargo test`), build tools (`tsc`, `cargo build`, `make`), linters, Docker, and package managers. [#315881](https://github.com/microsoft/vscode/issues/315881)

## May 12, 2026

* Add an idle-silence timer to the `run_in_terminal` tool that automatically promotes a sync command to background execution when it produces no output for a configurable period. [#315884](https://github.com/microsoft/vscode/issues/315884)

* Fix multi-line shell commands in the Agent Host terminal tool. [#312922](https://github.com/microsoft/vscode/issues/312922)

* Bundle a newer version of ConPTY (`conpty.dll`) directly with VS Code on Windows. [#224488](https://github.com/microsoft/vscode/issues/224488)

## May 11, 2026

* Add keyboard-interactive authentication support for Agent Host SSH connections. [#315588](https://github.com/microsoft/vscode/issues/315588)
