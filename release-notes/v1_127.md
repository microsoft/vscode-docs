## June 28, 2026

* Keep keyboard focus in the chat input when switching between sessions. [#323335](https://github.com/microsoft/vscode/issues/323335)

* Add a session header meta row with consistent pills and clickable diff and changes links. [#323334](https://github.com/microsoft/vscode/issues/323334)

* Add support for `/troubleshoot` for Agent Host Copilot CLI sessions. [#323317](https://github.com/microsoft/vscode/issues/323317)

## June 27, 2026

* Promote the integrated browser agent tools out of experimental. [#322540](https://github.com/microsoft/vscode/issues/322540)

* See which files have changed when using the Agent Host. [#318891](https://github.com/microsoft/vscode/issues/318891)

## June 26, 2026

* Add support for adding agent feedback without first making a selection. [#322696](https://github.com/microsoft/vscode/issues/322696)

* Run the Claude agent through an Anthropic subscription instead of Copilot quota (agent host only). [#314952](https://github.com/microsoft/vscode/issues/314952)

## June 25, 2026

* When a model's long-context tier has no additional cost, the UI no longer offers (or enforces) a smaller default tier and instead uses long context. [#322951](https://github.com/microsoft/vscode/issues/322951)

* Preserve the chat input draft when switching between chat sessions. [#315996](https://github.com/microsoft/vscode/issues/315996)

## June 24, 2026

* Configure built-in and extension tools for the Agent Host in Agent Customizations view. [#319812](https://github.com/microsoft/vscode/issues/319812)

* Add support for user-allowed permissions in the integrated browser, such as camera, location, and local network access. [#299521](https://github.com/microsoft/vscode/issues/299521)

## June 23, 2026

* Use pixel spinners for session list items instead of the static spinner codicon. [#321813](https://github.com/microsoft/vscode/issues/321813)

* Move the radio tower icon out of individual chat sessions to a global position. [#320114](https://github.com/microsoft/vscode/issues/320114)

* Update token pricing normalization to support a legacy “flat” pricing payload returned by some endpoints. [#318441](https://github.com/microsoft/vscode/issues/318441)
