## June 8, 2026

* Navigate to previous/next session in the sessions list using keyboard shortcuts. _[#320331](https://github.com/microsoft/vscode/issues/320331)_

* Remove the option to run the Copilot CLI Agent Host in empty VS Code windows and with the worktree option when a workspace is open. Both cases redirect to the Agents window. _[#318811](https://github.com/microsoft/vscode/issues/318811)_

---

## June 7, 2026

* Add `setting(sessions.changes.openSingleFileDiff)` setting (Preview) to use single-file diff when selecting a file in the Changes tab. _[#320320](https://github.com/microsoft/vscode/issues/320320)_

* Add a chevron toggle button to the editor title bar in the Agents window to collapse or restore the secondary side bar. The button is visible in grid mode only. _[#320319](https://github.com/microsoft/vscode/issues/320319)_

* Show an animated progress indicator in the session header for in-progress and needs-input states, matching the sessions list indicator. _[#320318](https://github.com/microsoft/vscode/issues/320318)_

* Unified recency history powering the sessions picker and navigation. _[#320315](https://github.com/microsoft/vscode/issues/320315)_

* Restore all visible sessions and layout state in the Agents window on reload or reopen, including grid order, active session, pinned sessions, per-session layout, and auxiliary bar visibility. _[#320314](https://github.com/microsoft/vscode/issues/320314)_

---

## June 5, 2026

* WSL connection support in the Agents window. _[#307568](https://github.com/microsoft/vscode/issues/307568)_

* Customizable integrated browser toolbar. Show or hide individual toolbar actions via the context menu. _[#320016](https://github.com/microsoft/vscode/issues/320016)_ _[#318437](https://github.com/microsoft/vscode/issues/318437)_

* Add support for entering text in browser fields in a single agent tool call, reducing the number of steps previously needed to complete text input actions. _[#320182](https://github.com/microsoft/vscode/issues/320182)_

* Update the agent-aware terminal environment variable to `AI_AGENT` (value: `github_copilot_vscode_agent`) for cross-integrator standardization. _[#320127](https://github.com/microsoft/vscode/issues/320127)_

* Improve session list visual hierarchy. _[#320002](https://github.com/microsoft/vscode/issues/320002)_ _[#319372](https://github.com/microsoft/vscode/issues/319372)_

* Terminal steering messages use the session's configured model instead of the default model. _[#317059](https://github.com/microsoft/vscode/issues/317059)_

* Add support for the "Auto" model in the Agents window model picker. _[#311443](https://github.com/microsoft/vscode/issues/311443)_

---

## June 4, 2026

* Scope chat input history to the current session in the Agents window. Pressing `Up`/`Down` in the chat input only navigates prompts submitted in that same session, instead of including prompts from other sessions. _[#319341](https://github.com/microsoft/vscode/issues/319341)_

---

## June 2, 2026

* Multi-chat support for local sessions in the Agents window. _[#319602](https://github.com/microsoft/vscode/issues/319602)_

* Background send in the Agents window. Press `Alt+Enter` or `Alt`-click Send to start a session without navigating into it, so you can immediately compose your next message. _[#319598](https://github.com/microsoft/vscode/issues/319598)_

* Sessions grid keyboard navigation and bulk close. Use `Ctrl+1`–`Ctrl+9` (`Cmd+1`–`Cmd+9`) to focus sessions by position, and `Ctrl+K Ctrl+W` (`Cmd+K Cmd+W`) to close all sessions at once. _[#319596](https://github.com/microsoft/vscode/issues/319596)_

* URL history in the integrated browser. Press `Up`/`Down` in the address bar to navigate previously visited URLs. _[#303767](https://github.com/microsoft/vscode/issues/303767)_

* Create folder in the simple file dialog. You can now create a new folder directly from the simple file picker. _[#318768](https://github.com/microsoft/vscode/issues/318768)_

---

## June 1, 2026

* Regex flags for folding markers. Folding marker patterns in `language-configuration.json` now support a `{ pattern, flags }` object form, enabling flags like case-insensitive matching. _[#72989](https://github.com/microsoft/vscode/issues/72989)_
