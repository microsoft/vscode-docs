## June 4, 2026

* Scope chat input history to the current session in the Agents window. Pressing `Up`/`Down` in the chat input only navigates prompts submitted in that same session, instead of including prompts from other sessions. _[#319341](https://github.com/microsoft/vscode/issues/319341)_

---

## June 2, 2026

* Multi-chat support for local sessions in the Agents window. _[#319602](https://github.com/microsoft/vscode/issues/319602)_

* Background send in the Agents window. Press `Alt+Enter` or `Alt`-click Send to start a session without navigating into it, so you can immediately compose your next message. _[#319598](https://github.com/microsoft/vscode/issues/319598)_

* Sessions grid keyboard navigation and bulk close. Use `Ctrl+1`–`Ctrl+9` (`Cmd+1`–`Cmd+9`) to focus sessions by position, and `Ctrl+K Ctrl+W` (`Cmd+K Cmd+W`) to close all sessions at once. _[#319596](https://github.com/microsoft/vscode/issues/319596)_

---

## June 1, 2026

* Regex flags for folding markers. Folding marker patterns in `language-configuration.json` now support a `{ pattern, flags }` object form, enabling flags like case-insensitive matching. _[#72989](https://github.com/microsoft/vscode/issues/72989)_
