## January 12, 2026

* Terminal command output now streams inline inside the Chat view. The terminal output auto-expands on command execution and collapses on success. [#257468](https://github.com/microsoft/vscode/issues/257468)

* Adds a trash icon to the Hidden Terminals menu item that deletes all hidden chat terminals at once. [#275235](https://github.com/microsoft/vscode/issues/275235)

* Adds a new "Match File Path Case" toggle to the Search view's "files to include" input, allowing users to control whether file paths and glob patterns should be matched case-sensitively. [#10633](https://github.com/microsoft/vscode/issues/10633)

## January 11, 2026

* The terminal now supports the kitty keyboard protocol (CSI u), enabling more sophisticated keyboard input handling and providing access to previously unavailable key combinations. [#286809](https://github.com/microsoft/vscode/issues/286809)

* The terminal now supports win32-input-mode, improving keyboard handling compatibility with Windows console applications. [#286896](https://github.com/microsoft/vscode/issues/286896)

* The terminal now supports SGR 221 and 222 escape sequences, allowing independent control of bold and faint text attributes for more granular formatting control. [#286810](https://github.com/microsoft/vscode/issues/286810)

* Improved discoverability of archived chat sessions in the Chat view, making it easier to locate and access previously archived conversations. [#286815](https://github.com/microsoft/vscode/issues/286815)

## January 8, 2026

* The terminal tool now supports a `timeout` parameter to control how long terminal commands run before timing out, preventing unnecessary polling commands. [#286598](https://github.com/microsoft/vscode/issues/286598)

* Updated the list of npm commands that are safe for automatic execution. [#286463](https://github.com/microsoft/vscode/issues/286463)

* The terminal suggest toolbar no longer shows the selection mode option when quick suggestions are disabled, reducing confusion. [#286440](https://github.com/microsoft/vscode/issues/286440)

* The `terminal.integrated.suggest.quickSuggestions` setting can now be configured in the Settings editor instead of requiring manual JSON editing. [#286075](https://github.com/microsoft/vscode/issues/286075)

* Added a `workbench.mcp.startServer` command to start a specific or all MCP servers to discover their tools. [#283959](https://github.com/microsoft/vscode/issues/283959)

* Exploration work on rich integrated web browser using WebContentsView or controlledframe to overcome limitations of the current iframe-based Simple Browser. [#277298](https://github.com/microsoft/vscode/issues/277298)

* A new proposed API enables extensions to show buttons inline after the input box in quick inputs, in addition to the title bar location. [#221397](https://github.com/microsoft/vscode/issues/221397)

## January 7, 2026

* You can now import a chat session into the Chat view instead of only opening it in a new editor tab. [#283954](https://github.com/microsoft/vscode/issues/283954)

* Fixed an issue where auto approval information for the fetch tool was not visible when hovering over the tool call. [#282238](https://github.com/microsoft/vscode/issues/282238)

* Added built-in support for MCP Apps, enabling servers to provide custom UI for tool invocation. [#260218](https://github.com/microsoft/vscode/issues/260218)

## December 28, 2025

* The quick input now supports overflow buttons, enabling secondary actions to be placed in an overflow menu. [#285213](https://github.com/microsoft/vscode/issues/285213)
