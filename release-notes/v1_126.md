## June 21, 2026

* Add a `setting(chat.agent.allowBypassApprovals)` setting to control whether the agent can bypass tool approval prompts. [#315869: Support Disabling Bypass Approvals](https://github.com/microsoft/vscode/issues/315869)

## June 20, 2026

* Rename the agent host Copilot CLI session type to "Copilot" in the session type picker. [#322193: Rename AH session type to 'Copilot'](https://github.com/microsoft/vscode/issues/322193)

## June 19, 2026

* Add a "Don't show again" option to Copilot usage banners. [#321949: Add "Don't show again" option for Copilot usage banners](https://github.com/microsoft/vscode/issues/321949)

* Add new codicons to indicate PR CI failures and PR comments in the chat session list. [#321565: Codicon Request: PR failed CI & PR has comments icon](https://github.com/microsoft/vscode/issues/321565)

## June 18, 2026

* Add session-level token usage data to give insight into total token consumption and cache hit rate for a chat session. [#315283: Provide session level token usage data](https://github.com/microsoft/vscode/issues/315283)
* Deprecate the built-in chat Edit mode and remove the `setting(chat.editMode.hidden)` setting. [#321802: Deprecate built-in chat "Edit" mode](https://github.com/microsoft/vscode/issues/321802)
* Update chat session list items to use the pixel spinner, aligning with the Agents window. [#321813: Update workbench chat session list items to use pixel spinners](https://github.com/microsoft/vscode/issues/321813)

## June 16, 2026

* Unify the context and reasoning effort pickers into a single model customization picker. [#320816](https://github.com/microsoft/vscode/issues/320816)

## June 15, 2026

* Add support for inline renaming of chats directly in their tab (Agents window). [#321386](https://github.com/microsoft/vscode/issues/321386)
