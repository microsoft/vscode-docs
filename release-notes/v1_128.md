## Workbench

### Quick open file ranges

Quick Open (`kb(workbench.action.quickOpen)`) now supports a line range suffix. Search text can be followed by `:<startLine>-<endLine>` (for example `file.txt:20-40`), and the file opens with that range selected. Column positions are also supported, such as `file.txt:20:3-40:5`.

## June 30, 2026

* Add support for configuring OpenTelemetry (OTel) settings through enterprise policy, letting organizations enable telemetry collection for their users. [#298965](https://github.com/microsoft/vscode/issues/298965)

## June 29, 2026

* Add support for custom endpoint model options such as `temperature`, so BYOK models work with providers that enforce strict parameter requirements (for example, Moonshot/Kimi). [#321514](https://github.com/microsoft/vscode/issues/321514)

* Rename the tour's **Skip** button to **End Tour** and add an `Esc` key hint to make dismissing the entire tour clearer. [#323446](https://github.com/microsoft/vscode/issues/323446)

## June 26, 2026

* Add support for running the Claude agent through your own Anthropic API credentials (BYOK) instead of consuming GitHub Copilot quota. [#314952](https://github.com/microsoft/vscode/issues/314952)
