## April 26, 2026

* The Agents app supports sharing single sign-on (SSO) with Visual Studio Code on Windows. The SSO is bidirectional, and sign-out propagates between both apps. [#308028](https://github.com/microsoft/vscode/issues/308028), [#312624](https://github.com/microsoft/vscode/issues/312624)

## April 24, 2026

* Add a skill tool for agent customizations, along with support for `context: fork`. [#312444](https://github.com/microsoft/vscode/issues/312444)

* Add accessibility improvements to the question carousel for terminal actions. [#312423](https://github.com/microsoft/vscode/issues/312423)

* Add support for Claude Code agent sessions in the Agents app. [#312416](https://github.com/microsoft/vscode/issues/312416)

* The Agents app shares workspace trust state with Visual Studio Code. [#308250](https://github.com/microsoft/vscode/issues/308250)

* Add support for the CP857 encoding in document encodings. [#300041](https://github.com/microsoft/vscode/issues/300041)

## April 23, 2026

* Adopt the Copilot CLI SDK session-title APIs as the source of truth for session names. [#311535](https://github.com/microsoft/vscode/issues/311535)

* Add keybindings such as `Ctrl+1` and `Ctrl+2` to quickly switch between sessions in the Agents app. [#310992](https://github.com/microsoft/vscode/issues/310992)

* Add auto model support to the Copilot CLI agent. [#311299](https://github.com/microsoft/vscode/issues/311299)

## April 22, 2026

* Add descriptions to the chat customization creator menu, helping users understand which skill location to use for each type of customization. [#295151](https://github.com/microsoft/vscode/issues/295151)

* Let users opt in to testing the TypeScript 7.0 nightlies. [#311966](https://github.com/microsoft/vscode/issues/311966)

* Copilot CLI responses in the chat panel show a model badge indicating which model handled the request. [#303960](https://github.com/microsoft/vscode/issues/303960)

* Add support for auto model selection in Copilot CLI. [#311779](https://github.com/microsoft/vscode/issues/311779)

## April 21, 2026

* The Copilot CLI SDK resolves `node-pty` from VS Code via `hostRequire`, eliminating the need to copy `node-pty` binaries into the SDK's prebuilds folder at build time or runtime. [#307746](https://github.com/microsoft/vscode/issues/307746)
