## April 17, 2026

* Updated **Run VS Code command** agent tool to add support for allowlisting specific commands and narrower approvals. [#302411](https://github.com/microsoft/vscode/issues/302411)

## April 16, 2026

* Add support for sorting sessions in the Agent Sessions view by **Created** or **Updated** date. [#308566](https://github.com/microsoft/vscode/issues/308566)

* You can edit queued chat messages via a new **Edit** action in the queued message context menu. [#308552](https://github.com/microsoft/vscode/issues/308552)

* Background terminal command completion is surfaced as a system notification rather than only as inline text in the chat response, improving accessibility and discoverability. [#308549](https://github.com/microsoft/vscode/issues/308549)

## April 15, 2026

* Add support for self-updating the Agents app on macOS. [#308646](https://github.com/microsoft/vscode/issues/308646)

* Copilot CLI sessions indicate whether they were created by VS Code or externally. [#308543](https://github.com/microsoft/vscode/issues/308543)

* In `package.json` files, the dependency hover shows the currently installed version alongside the latest published version of a package. [#307648](https://github.com/microsoft/vscode/issues/307648)

* Images in JSDoc comments, including `<img>` HTML tags, render correctly in hovers, completion details, and signature help for JavaScript and TypeScript files. [#231792](https://github.com/microsoft/vscode/issues/231792)

## April 14, 2026

* The Autopilot permission mode persists across sessions. You can configure the default permission level with the `setting(chat.permissions.default)` setting. [#309562](https://github.com/microsoft/vscode/issues/309562)

* When an agent sends input to a terminal, the terminal output is automatically included in the result after a brief delay, saving an extra agent turn. [#309509](https://github.com/microsoft/vscode/issues/309509)

* Add support for auto-approve session configuration in Agent Host with three modes: Default Approvals, Bypass Approvals, and Autopilot (Preview). [#309337](https://github.com/microsoft/vscode/issues/309337)

* Add support for switching back to the main window from auxiliary (floating) windows. [#306571](https://github.com/microsoft/vscode/issues/306571)

* Add support for subagents and agent teams in the Agent Host Protocol. [#305755](https://github.com/microsoft/vscode/issues/305755)

## April 13, 2026

* Add support for worktree and git isolation in Agent Host sessions. [#305325](https://github.com/microsoft/vscode/issues/305325)

* Copilot CLI, Claude Code, and Gemini CLI are recognized as shell types in the terminal. [#290830](https://github.com/microsoft/vscode/issues/290830)

## April 12, 2026

* Copilot CLI generates meaningful branch names based on the user's prompt when creating worktrees for background agent sessions. [#306191](https://github.com/microsoft/vscode/issues/306191)
