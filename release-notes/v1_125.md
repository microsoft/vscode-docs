## June 10, 2026

* Add a folder picker for Agent Host sessions in multi-root windows, so you can choose which workspace folder a session targets. [#318069](https://github.com/microsoft/vscode/issues/318069)

## June 9, 2026

* Add the `/chronicle` command set to the Agent Host, providing session history insights — `standup`, `search`, `tips`, `cost-tips`, `improve`, and `reindex` — directly from chat. [#320648](https://github.com/microsoft/vscode/issues/320648)

* Improve the display of file paths in the Agent Host for better readability. [#316541](https://github.com/microsoft/vscode/issues/316541)

## June 8, 2026

* Enhance the Cache Explorer view to make multi-agent sessions easier to understand and navigate, and to surface more detailed prompt-signature allocation information. [#320137](https://github.com/microsoft/vscode/issues/320137)

* Add support for qualified tool names in tool sets. [#271589](https://github.com/microsoft/vscode/issues/271589)
