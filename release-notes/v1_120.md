## May 6, 2026

* Hide archived sessions by default in the chat sessions quick pick to reduce visual clutter. Searching in the quick pick still matches archived sessions, so you can revive one by title. _[#297421](https://github.com/microsoft/vscode/issues/297421)_

## May 5, 2026

* Agent host terminals respect the user's preferred shell from `setting(terminal.integrated.defaultProfile)` instead of hardcoding the platform default. _[#313160](https://github.com/microsoft/vscode/issues/313160)_
* Add support for custom diff editors with the new `customDiffEditorProvider` proposed API. Extensions that implement `resolveCustomDiffEditor` can render a unified diff in a single webview with access to both the original and modified documents, instead of two side-by-side custom editor webviews. _[#298924](https://github.com/microsoft/vscode/issues/298924)_
* Restore the ability to select and copy code in chat edit suggestions that the agent proposes to remove. _[#252762](https://github.com/microsoft/vscode/issues/252762)_

## May 4, 2026

* Add a custom duration option to the **Snooze Inline Suggestions** Quick Pick. _[#288770](https://github.com/microsoft/vscode/issues/288770)_
* Reduce the CSS context attached when adding a DOM element to chat. The attachment only includes modified computed styles and CSS rules and variables that actually affect the element. _[#314188](https://github.com/microsoft/vscode/issues/314188)_
