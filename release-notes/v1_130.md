## July 15, 2026

* Set `agents.voice.handsFree` to be opt-in so voice mode starts in manual mode unless you enable hands-free. _[#326045: set hands free mode to false by default for voice mode](https://github.com/microsoft/vscode/issues/326045)_

## July 16, 2026

* Add support for dictating prompts directly from the chat input in the Agents app with on-device live transcription. _[#317172: Agents App: Add a voice to text option](https://github.com/microsoft/vscode/issues/317172)_
* Improve dictation quality by improving recognition for specialized terms and reducing filler words in transcribed prompts. _[#315508: Feature Request: Improve VSCode Speech Input Accuracy and Polishing for Specialized Terms and Conversational Expressions](https://github.com/microsoft/vscode/issues/315508)_

## July 17, 2026

* Add support for returning `MethodNotFound` JSON-RPC errors for unsupported Agent Host requests before initialization, which avoids silent client timeouts during handshake failures. _[#326322: Agent Host: Return an error for unsupported requests before initialize](https://github.com/microsoft/vscode/issues/326322)_

## July 18, 2026

* Add support for running automations without an open workspace by selecting a **No workspace** target in the Agents window. _[#326330: Automations: support workspace-less automations](https://github.com/microsoft/vscode/issues/326330)_

## July 19, 2026

* Add support for programmatically requesting semantic token data from semantic highlighting providers. _[#91375: Define a `vscode.executeSemanticHighlightProvider` command](https://github.com/microsoft/vscode/issues/91375)_

## July 20, 2026

* Add support for showing timestamps in chat messages, which helps you understand how long prompts and responses are taking. _[#268829: Copilot Chat feature request: show timestamp for messages/prompts](https://github.com/microsoft/vscode/issues/268829)_
* Add support for showing per-file `+X -Y` diff stats in each file header in the single-pane Changes editor. _[#326563: Show per-file diff stats in the single-pane Changes editor headers](https://github.com/microsoft/vscode/issues/326563)_
* Add support for adding line-specific feedback from the diff gutter by hovering a line number in Agents window diff views. _[#326564: Add compact 'Add Feedback' action on hovered diff line numbers in the Agents window](https://github.com/microsoft/vscode/issues/326564)_
* Refine quick-chat rows in the Sessions list to use a compact single-line layout while regular sessions keep the two-line layout. _[#326565: Refine quick-chat row rendering in the Sessions list (compact single-line rows)](https://github.com/microsoft/vscode/issues/326565)_
* Improve Copilot Auto explainability by showing the selected model and complexity context when Auto routes a request. _[#321602: Explainability for Auto](https://github.com/microsoft/vscode/issues/321602)_
