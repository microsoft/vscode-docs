## March 7, 2026

* The modal editor dialog can now be resized and moved within the workbench window, making it easier to work with on larger displays. This change is per session and is not persisted across restarts. [#293915](https://github.com/microsoft/vscode/issues/293915)

## March 6, 2026

* The integrated browser now has a right-click context menu with standard browser actions and an **Add Element to Chat** option. [#292908](https://github.com/microsoft/vscode/issues/292908)

* A new `setting(editor.find.closeOnResult)` setting enables the Find dialog to automatically close after a search match is found, moving focus directly to the matched text in the editor. This setting is disabled by default. [#264818](https://github.com/microsoft/vscode/issues/264818)

## March 4, 2026

* The Claude agent mode is no longer in preview and is now generally available. [#290048](https://github.com/microsoft/vscode/issues/290048)
