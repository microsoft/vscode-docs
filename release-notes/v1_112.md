# Visual Studio Code 1.112

_Release date: March 18, 2026_

<!-- DOWNLOAD_LINKS_PLACEHOLDER -->

---

Welcome to the 1.112 release of Visual Studio Code. This release includes various **improvements across agent management and the developer experience**.

* [**Copilot CLI integration**](#copilot-cli-integration): work with MCP servers in the CLI, and seamlessly continue CLI sessions in VS Code.

* [**Integrated browser**](#integrated-browser): debug web apps, use context menus, and zoom in the built-in browser.

* [**Agent plugins**](#agent-plugins): manage plugins per workspace and install individual plugins from a repository.

* [**MCP auto-allow configuration**](#mcp-auto-allow-configuration): mark MCP servers and tools as enabled or auto-allowed directly in `mcp.json`.

* [**Chat customizations discovery**](#chat-customizations-discovery): discover customization files from parent repositories and configure hooks at the user level.

* [**Chat UX improvements**](#chat-ux-improvements): browse image carousels in responses, let agents read images from disk, and export agent debug logs.

Happy Coding!

---

VS Code is rolling out gradually to all users. Use **Check for Updates** in VS Code to get the latest version immediately.

To try new features as soon as possible, [**download the nightly Insiders build**](https://code.visualstudio.com/insiders), which includes the latest updates as soon as they are available.

## Terminal

### Improved IME composition for the terminal

When typing with an Input Method Editor (IME) near the right edge of the terminal, the composition preview text could previously overflow outside the terminal boundaries. The composition view is now constrained to the available space between the cursor and the terminal's right edge. As new characters are entered, older characters are progressively hidden, keeping the preview text within the terminal viewport. This matches the behavior of other modern terminals such as Ghostty.
