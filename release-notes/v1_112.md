## March 16, 2026

* Copilot CLI background agents now support MCP servers. [#282590](https://github.com/microsoft/vscode/issues/282590)

## March 14, 2026

* A new image carousel view lets you browse and navigate images from chat sessions and workspace files. Images from tool calls are rendered as clickable pills, and the carousel editor supports previous/next navigation across multiple images. [#301606](https://github.com/microsoft/vscode/issues/301606)

## March 13, 2026

* File links in Copilot CLI sessions are now clickable and open the referenced file directly in the VS Code editor. [#296440](https://github.com/microsoft/vscode/issues/296440)

## March 12, 2026

* Add support for plugin-specific component paths (commands/skills/agents/hooks/MCP) per the updated plugins spec, and introduces an "Install Plugin from Source" flow to clone and install plugins directly from a GitHub repo / git URL. [#300945](https://github.com/microsoft/vscode/issues/300945)

* Discover chat customizations in parent folders if the current workspace folder is not a Git repository (`chat.useCustomizationsInParentRepositories` setting). [#293277](https://github.com/microsoft/vscode/issues/293277)

## March 11, 2026

* The minimap right-click context menu now includes a **Side** option, improving discoverability of minimap positioning settings. [#299653](https://github.com/microsoft/vscode/issues/299653)

* Add a new default location for hook configurations in `~/.copilot/hooks`. [#296793](https://github.com/microsoft/vscode/issues/296793)

* When an agent asks a clarifying question with predefined options, you can now always type a freeform text response in addition to selecting from the listed choices. [#300922](https://github.com/microsoft/vscode/issues/300922)

* The integrated browser now supports independent zoom controls, letting you zoom in and out of the browser content without affecting the VS Code editor zoom level. Zoom is accessible via the browser menu and keyboard shortcuts. [#300849](https://github.com/microsoft/vscode/issues/300849)

## March 10, 2026

* Added support for chat to directly analyze images stored on disk. [#296817](https://github.com/microsoft/vscode/issues/296817)

* Reserved context in the chat interface now displays with distinct visual treatment in the context usage indicator, making it visually distinct from regular context. [#295110](https://github.com/microsoft/vscode/issues/295110)

* Added VS Code update notification support for layouts where the Status Bar is hidden, so users can access update information from the title bar or a notification banner. [#293891](https://github.com/microsoft/vscode/issues/293891)

* Added support for steering and queuing messages in Copilot CLI agents. [#300304](https://github.com/microsoft/vscode/issues/300304)

* Add support for permission levels for Copilot CLI sessions. [#300303](https://github.com/microsoft/vscode/issues/300303)

## March 9, 2026

* Agent plugins can be enabled or disabled at the workspace and global level. [#300271](https://github.com/microsoft/vscode/issues/300271)

* MCP servers can be enabled or disabled at the workspace and global level. [#243620](https://github.com/microsoft/vscode/issues/243620)

## March 7, 2026

* The modal editor dialog can now be resized and moved within the workbench window, making it easier to work with on larger displays. This change is per session and is not persisted across restarts. [#293915](https://github.com/microsoft/vscode/issues/293915)

## March 6, 2026

* The integrated browser now has a right-click context menu with standard browser actions and an **Add Element to Chat** option. [#292908](https://github.com/microsoft/vscode/issues/292908)

* A new `setting(editor.find.closeOnResult)` setting enables the Find dialog to automatically close after a search match is found, moving focus directly to the matched text in the editor. This setting is disabled by default. [#264818](https://github.com/microsoft/vscode/issues/264818)

## March 4, 2026

* The Claude agent mode is no longer in preview and is now generally available. [#290048](https://github.com/microsoft/vscode/issues/290048)
