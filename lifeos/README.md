## Getting Started

Clone the repo and install dependencies:
```bash
git clone https://github.com/IBERMOLINA/LifeOS.git
cd LifeOS
npm install
npm start
```

Or run with Docker:
```bash
docker build -t lifeos .
docker run -p 3000:3000 lifeos
```
