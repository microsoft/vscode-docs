# Core Editor Features

Visual Studio Code comes with a rich set of built-in features for productive code editing, debugging, and source control. This section covers the core capabilities available out of the box.

## Configure

Customize VS Code to match your workflow. Set up [keyboard shortcuts](/docs/configure/keybindings), adjust [settings](/docs/configure/settings), manage [extensions](/docs/configure/extensions/extension-marketplace), and create [profiles](/docs/configure/profiles) for different projects.

## Edit code

Take advantage of powerful editing features like [IntelliSense](/docs/editing/intellisense) for smart completions, [code navigation](/docs/editing/editingevolved) to quickly move through your codebase, [refactoring](/docs/editing/refactoring) tools, and [snippets](/docs/editing/userdefinedsnippets).

## Build, debug, and test

Run [tasks](/docs/debugtest/tasks) to automate your build process, use the built-in [debugger](/docs/debugtest/debugging) with support for many languages, and run your [tests](/docs/debugtest/testing) directly from the editor.

## Source control

VS Code has integrated [Git support](/docs/sourcecontrol/overview) with features for [staging and committing](/docs/sourcecontrol/staging-commits), [branch management](/docs/sourcecontrol/branches-worktrees), [resolving merge conflicts](/docs/sourcecontrol/merge-conflicts), and [collaborating on GitHub](/docs/sourcecontrol/github).

## Terminal

Use the [integrated terminal](/docs/terminal/basics) to run commands without leaving the editor. Configure [terminal profiles](/docs/terminal/profiles), take advantage of [shell integration](/docs/terminal/shell-integration), and customize the [terminal appearance](/docs/terminal/appearance).
